Sauvegarde automatique dans votre Google Drive. Configurez votre Client ID dans Paramètres.

Comment obtenir un Client ID Google :
1. Allez sur console.cloud.google.com
2. Créez un projet → APIs → Google Drive API → Identifiants
3. Créez un ID client OAuth 2.0 (type: Application Web)
4. Collez l'ID dans Paramètres → Client ID Google
